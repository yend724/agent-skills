# Agent Skills

[English](./README.md) | 日本語

このリポジトリは [yend724](https://github.com/yend724) が公開している Agent Skills 集です。

スキルは [Agent Skills](https://agentskills.io/home) のオープンフォーマットに準拠しており、Claude Code、VS Code with GitHub Copilot、Cursor、Gemini CLI、OpenAI Codex などの対応エージェントで利用できます。

## インストール

このリポジトリのスキルは `npx skills add` で追加できます。

```bash
npx skills add yend724/agent-skills
```
